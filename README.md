## Note varie
Aggiungere un pacchetto: prima aggiungerlo in package.json > dependences, poi da bash
```console
    npm install npm-install-missing
```

## Avviare client e server
client
```console
    npm run serve
```

server
```console
    cd server && npm run devserver
```
